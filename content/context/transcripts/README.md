# transcripts/

Drop raw transcripts here — videos, voice notes, recorded calls, anything where you or
the team speak naturally.

Claude reads these **before writing** to learn the natural speaking voice of the author.
This is one of the strongest ways to make written content sound like a real person and
not generic AI text.

- One transcript per file is fine.
- Raw and messy is fine. No need to clean them up.
- Use the project's primary language where possible.
