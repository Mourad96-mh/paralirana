# content/ — Content production folder

This is where all content is created and published.

**Before writing anything, Claude must read everything in `context/`** — this defines
the brand voice, the style rules, and the writing examples.

## Workflow

1. Read all files in `context/` (voice, style, examples, transcripts)
2. Pull the keyword brief and topical map from `../research/`
3. Write the draft into `drafts/`
4. Save any images into the relevant `images/` subfolder
5. Once a draft is approved, move it to `published/`

## Folders

- `context/` — voice, style guide, writing examples, transcripts. **Read first.**
- `drafts/` — work in progress
- `published/` — approved, final content
- `images/generated/` — AI-generated images
- `images/screenshots/` — screenshots
- `images/assets/` — logos, icons, other assets

## Language

Primary content language is set in `CLAUDE.md`.
